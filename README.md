# 🕐 Gaokao Countdown

简洁高效的高考倒计时网页应用，专为高中班级一体机设计，支持多种倒计时场景和个性化主题。

## ✨ 主要功能

| 功能 | 说明 |
|------|------|
| ⏱️ **当前时间** | 实时显示当前时间和日期 |
| 📅 **高考倒计时** | 精确到秒的高考倒计时 |
| 📋 **课表倒计时** | 预设小练、考试、晚读等特殊时段，自动启动倒计时 |
| ⏲️ **自定义倒计时** | 手动设置任意时长的倒计时，暂停、重置、删除 |
| 🎨 **主题系统** | 支持VS Code、赛博朋克、水墨古风、绪山真寻等预设主题 |
| 📁 **文件夹主题** | 导入/导出完整主题包，包括背景图和所有配置 |
| 📐 **布局调整** | 3种预设布局 + 元素微调功能 |
| 📱 **全屏模式** | 简洁的全屏显示，5秒无操作自动隐藏控制栏 |

## 🚀 快速开始

### 基本使用

直接用浏览器打开 `index.html` 即可使用。

**推荐使用 Chrome/Edge 浏览器** 以获得最佳体验。

### 默认配置

- 默认主题：VS Code 深色风格
- 默认课表：工作日的小练、考试、晚读时段
- 默认全屏：点击右下角全屏图标进入全屏

## 🎨 主题使用

### 预设主题

| 主题 | 说明 |
|------|------|
| VS Code | 经典 VS Code 深色风格，护眼舒适 |
| 赛博朋克 | 霓虹配色，未来感十足 |
| 水墨古风 | 中国风配色，典雅大气 |
| 绪山真寻 | 动漫风格，明亮清新 |
| Windows11 | Windows 11 Fluent Design 风格 |

### 文件夹主题（推荐）

#### 创建自定义主题

1. 复制 `assets/themes/ExamplesTopic/` 文件夹
2. 重命名为你喜欢的主题名称（例如 `MyTheme`）
3. 修改 `config.json` 文件配置颜色、字体等
4. 可选：添加背景图到文件夹
5. 可选：编辑 `README.md` 记录主题说明

#### config.json 配置

```json
{
  "name": "主题名称",
  "author": "作者名",
  "version": "1.0",
  "theme": {
    "primary": "#007acc",
    "bg-primary": "#1e1e1e",
    "bg-secondary": "#252526",
    "text-primary": "#cccccc",
    "text-secondary": "#808080",
    "accent-color": "#007acc",
    "border-color": "#3c3c3c"
  },
  "backgroundImage": "bg.jpg"
}
```

#### 导入/导出

1. 在个性化面板点击「下载主题模板」获取 ExamplesTopic
2. 修改后点击「导入文件夹主题」导入
3. 或者直接复制主题文件夹到 `assets/themes/`

## 📐 布局调整

### 预设布局

| 布局 | 说明 |
|------|------|
| 紧凑 | 元素间距 5px，适合屏幕有限的场景 |
| 默认 | 标准间距 15px，平衡舒适 |
| 宽松 | 间距 30px，适合大屏展示 |

### 元素微调

选择要调整的元素后：
- **向上/向下** - 调整 margin-top
- **高亮/暗化** - 调整 opacity（不透明度）
- **重置** - 清除所有自定义样式

2. 添加/编辑/删除时段

## 🔧 浏览器要求

- Chrome 86+ (推荐)
- Edge 86+
- Firefox/其他：传统单文件导入/导出可用，文件夹功能受限

### File System Access API（推荐）

文件夹导入/导出功能需要 File System Access API：
- 必须在 HTTP/HTTPS 环境运行（`localhost` 也可以）
- `file://` 协议会降级为传统文件选择

## 💡 使用技巧

### 班级一体机部署

1. 将整个 Clock 文件夹复制到一体机
2. 设置浏览器主页为 `index.html`
3. 根据班级习惯调整课表和主题
4. 导出主题/配置文件作为备份

### 性能优化

- 不用的时段从课表中删除
- 背景图使用合适分辨率（推荐 1920x1080）
- 关闭不必要的动画效果

## 📝 注意事项

- 配置和设置都保存在浏览器的 localStorage
- 清除浏览器数据会丢失个性化设置
- 定期导出配置作为备份
- 导入配置会覆盖现有设置

## 🤝 开发

### 模块化架构

项目使用 IIFE（立即执行函数）进行模块化封装，避免全局变量污染。

```javascript
const ClockCore = (function() {
    'use strict';
    // 模块代码
    return { /* 公开API */ };
})();
```

### 可用事件

| 事件 | 参数 | 说明 |
|------|------|------|
| stateChange | { key, value } | 状态变更 |
| layoutChanged | { presetName } | 布局变更 |
| themeApplied | { themeName } | 主题应用 |
| elementStyleChanged | { elementKey, styles } | 元素样式变更 |
| configImported | { config } | 配置导入 |
